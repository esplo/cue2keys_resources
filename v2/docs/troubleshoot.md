# トラブルシューティングガイド

ここに記載のないトラブルが発生した場合は、お気軽にご連絡ください。追記していきます。

## モジュールを認識しない

適宜ペンダントをPCから外し、再度接続してみてください。

まず、問題が起きているモジュールを特定するため、モジュールを着脱して`問題が起こる最小の組み合わせ`を作りましょう。
数が多い場合は後回しでも良いですが、困ったら原因を絞り込むために着脱して確認してみましょう。

### 4pinケーブルの確認

4pinケーブルの接続が外れかかっている場合が多いです。

特にキーモジュールは移動中に引っ張られて抜けてしまうことがあります。
1番目のキーをひっくり返し、後ろにある隙間からマイナスドライバーなどで4pinケーブルをコネクタ側に押し込んでください。
左右両方を少しずつ押し込むとしっかり挿せます。
その後、再度接続して確認してみてください。

### チャンネル（ハブ）の変更

同一チャンネルで同じアドレスを使用していると認識されなくなります。
別のハブにモジュールをつなぐ、または他のモジュールを外すなどして認識されるか確認してみてください。

### チャンネル0に繋いで確認

ペンダントのLED付近には4pin接続コネクタがあります。
これは特別なコネクタで、チャンネル0として扱われます。ここに接続したモジュールは操作はできませんが、アドレスの変更などが行え、他のチャンネルより優先的に認識されます。

ここに接続し、Webアプリの`その他`からチャンネル0のモジュールのアドレスを適当なものに更新してみてください。
その後、ハブに接続し、モジュールが認識されるか確認してみてください。

### 分解して確認

キーモジュールの場合は、キー間を接続しているFPCケーブルの接触も原因として考えられます。
[キーモジュールのガイド](./modules/keys.md)を参考に、分解して再度接続し確認してみてください。
